Installation of RAD edition

We recommend to use Composer in order to create a new project.

Once installed, run the following commands to create a new project:

{% highlight bash %} $ composer.phar create-project -s dev --prefer-dist --dev knplabs/rad-edition my_project $ cd my_project {% endhighlight %}

Installation of RAD bundle

If you want to use the bundle in the existing project, you can do the following:

{% highlight bash %} $ composer.phar require knplabs/rad-bundle@~2.3.2 {% endhighlight %} {% highlight php %} {% include bundle-installation/app-kernel.php %} {% endhighlight %} {% highlight yaml %} {% include bundle-installation/routing-dev.yml %} {% endhighlight %} {% highlight yaml %} {% include bundle-installation/routing.yml %} {% endhighlight %}
